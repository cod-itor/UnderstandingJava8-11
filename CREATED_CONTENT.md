## 🎬 Running the Examples

### Compile and Run (Java 8+)

```bash
# Navigate to Java8-11 folder
cd Java8-11

# Compile an example
javac LambdaExpressions/LambdaExpressions.java

# Run it
java -cp LambdaExpressions LambdaExpressions
```

### For Pattern Matching (Java 16+)

```bash
# Pattern Matching examples
javac --enable-preview PatternMatching/PatternMatchingExamples.java
java --enable-preview -cp PatternMatching PatternMatchingExamples
```

### Running Other Features

```bash
# forEach Examples
javac ForEachMethod/ForEachExamples.java
java -cp ForEachMethod ForEachExamples

# Optional Examples
javac OptionalClass/OptionalExamples.java
java -cp OptionalClass OptionalExamples

# Stream Examples
javac StreamAPI/StreamAPIExamples.java
java -cp StreamAPI StreamAPIExamples
```
